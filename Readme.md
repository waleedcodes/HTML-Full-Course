## 📚 Complete Curriculum

### 🏁 Chapter 1: HTML Foundations

- What is HTML and how websites really work
- Setting up your development environment (VS Code, browsers)
- Writing your very first webpage
- Understanding HTML structure: DOCTYPE, html, head, body
- **🔨 Mini Project:** Create a "Hello World" webpage

### 🧱 Chapter 2: Essential HTML Elements

- Headings (h1–h6) and paragraphs (p)
- Text formatting: b, i, u, mark, strong
- Creating clickable links (a href)
- Adding images (img src, alt)
- Building ordered and unordered lists (ul, ol, li)
- **🔨 Mini Project:** Build a basic resume with headings, image, and paragraphs

### 📦 Chapter 3: Semantic HTML Structure

- div vs span – block vs inline
- Semantic elements: header, footer, main, section, article, nav
- Why semantic HTML matters (SEO + accessibility)
- Best practices for structuring content
- **🔨 Mini Project:** Design a blog layout using semantic tags

### 🧩 Chapter 4: Forms & User Input

- Creating forms (form, action, method)
- Input types: text, email, password, checkbox, radio, date, number, file, color, range, url
- Using label, fieldset, legend
- Buttons and dropdowns (select, option, textarea)
- Placeholder, required, and validation patterns
- **🔨 Mini Project:** Create a signup form with validation

### 🎨 Chapter 5: Media Integration

- Adding audio (audio controls) and video (video controls)
- Embedding YouTube videos & Google Maps (iframe)
- Image optimization (alt text, responsive images)
- Creating multimedia-rich experiences
- **🔨 Mini Project:** Create a music album page with embedded media

### 📊 Chapter 6: Tables & Data

- Table structure (table, tr, th, td)
- Organizing data with thead, tbody, tfoot
- Advanced features: colspan, rowspan
- Styling and making tables accessible
- **🔨 Mini Project:** Build a class schedule or pricing table

### ⚙️ Chapter 7: Advanced HTML5 Features & Best Practices

- Global attributes: id, class, style, title, lang, data-\*
- Accessibility basics (alt text, aria attributes, semantic tags)
- Meta tags for SEO & social media sharing
- HTML entities & special characters
- Boilerplate HTML template for real projects

### 🌐 Chapter 8: HTML + CSS Basics

- Inline, internal, and external CSS
- Using class and ID for styling
- Basic styling examples (colors, fonts, spacing)
- Linking CSS files correctly
- **🔨 Mini Project:** Create a simple landing page with basic styling

### 🏆 Final Project: Portfolio Website

- Plan your multi-page portfolio structure
- Build Home, About, Projects, and Contact pages
- Add navigation menu and form functionality
- Embed media, use tables, and apply semantic structure
- Prepare your site for deployment

## 🚀 Getting Started

1. **Clone this repository**

   ```bash
   git clone https://github.com/waleedcodes/HTML-Full-Course.git
   cd HTML-Full-Course
   ```

2. **Install VS Code** (if not already installed)

   - Download from [code.visualstudio.com](https://code.visualstudio.com)

3. **Start with Chapter 1**

   - Open the `chapter-1` folder
   - Follow along with the video tutorials

4. **Build each project**
   - Complete the mini projects in each chapter
   - Experiment with your own modifications
